# Batalha Naval em Java

Projeto desenvolvido em Java utilizando Programação Orientada a Objetos.

## Funcionalidades

- Mapa 10x10
- Jogador vs Computador
- Jogador vs Jogador
- Posicionamento manual de barcos
- Posicionamento automático de barcos
- Ataques por coordenadas
- Verificação de vitória

## Tecnologias

- Java
- IntelliJ IDEA

## Como executar

1. Abrir o projeto no IntelliJ IDEA
2. Executar a classe Main.java
3. Escolher o modo de jogo
